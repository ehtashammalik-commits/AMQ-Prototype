OpenAI API Key Required

To use the transcription feature, you need to provide your OpenAI API key. Your key will be stored locally in your browser.

🔐 Security Note: Your API key is stored only in your browser's localStorage and is never sent to any server except OpenAI.

How to get an API Key:

  1. Go to OpenAI API Keys
  2. Click "Create new secret key"
  3. Copy and paste it here
