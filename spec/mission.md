# Mission

## Purpose
Build a QM (Quality Management) transcription mockup application that automatically transcribes pre-recorded audio calls in Arabic and populates QM evaluation forms based on transcription analysis.

## Vision
Create a fully functional prototype that mirrors ExpertFlow's QM interface, enabling quality managers to review recorded customer service calls with automated transcription, silence detection, and form population.

## Core Objectives
- Provide conversation view with audio playback for pre-recorded calls
- Automatically transcribe Arabic audio using OpenAI API
- Detect silence periods (>10 seconds) in recordings
- Auto-populate QM forms based on transcription analysis
- Match ExpertFlow's visual design and user experience
- Support real audio files with playable controls

## QM Evaluation Criteria (Auto-detected)
- Initial Greeting verification
- Silence Detection (>10 seconds)
- Request for Location Details
- Request for Incident Details
- Request for Medical/Emergency Information
- Requirement Clarity Assessment

## Success Metrics
- Accurate Arabic transcription
- Correct silence detection
- Accurate form auto-population
- UI parity with ExpertFlow screenshot
- Fully functional within 2-hour delivery window
