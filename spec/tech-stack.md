# Tech Stack

## Frontend
- **Framework**: Angular 18+ (Core framework)
- **Styling**: SCSS/SASS (matching ExpertFlow design)
- **UI Components**: Angular Material (optional) / Custom components
- **HTTP Client**: Angular HttpClient
- **State Management**: Angular Services with RxJS

## Transcription & Audio
- **Transcription API**: OpenAI Whisper API
- **Audio Playback**: HTML5 Audio API
- **Audio Processing**: Web Audio API (for silence detection)

## Core Features
- **Conversation Management**: Angular Services
- **QM Form Analysis**: Custom TypeScript logic
- **Silence Detection**: Web Audio API analysis
- **Data Models**: TypeScript interfaces

## Development Tools
- **Package Manager**: npm
- **Runtime**: Node.js 18+
- **Build Tool**: Angular CLI / Webpack
- **Code Formatting**: Prettier
- **Development Server**: Angular CLI dev server (ng serve)

## External APIs
- **OpenAI API** (for Arabic transcription via Whisper)
- **No backend server required** (Standalone prototype)

## Color Scheme (from ExpertFlow)
- Primary: ExpertFlow blue (#0066CC or similar)
- Secondary: Light gray backgrounds
- Accent: Orange/amber for highlights
- Text: Dark gray for readability
